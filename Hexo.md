

# Hexo

## 博客目录构成介绍

- node_modules
- public
- scaffolds
- source
  - _posts
  - about
  - archive
  - img
  - tags

> Node_modules是node.js各种库的目录，public是生成的网页目录，scaffolds，储存着新文章新页面的初始位置，source存放着文章（_posts)、各类页面、图像等文件，themes存放着主体文件，一般用不到。

```json
title: 文章标题
catalog: 是否显示段落目录
date: 文章日期
subtitle: 子标题
header-img: 顶部背景图片
top: 是否置顶
tags: 标签
categories: 分类
```
